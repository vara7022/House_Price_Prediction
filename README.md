# House_Price_Prediction
Problem Statement

Predict house prices based on property features such as size, number of bedrooms, and age of the house.

Dataset Description

The dataset contains 15 records with the following features:

Size_sqft

Bedrooms

Age

Price (Target variable)

Model Used

Linear Regression

Results

Model performance was evaluated using:

Mean Absolute Error (MAE)

R² Score

Conclusion

House size and price per square foot significantly influence price prediction. The experiment also demonstrated the importance of proper train-test splitting to avoid overfitting.
